## 適用統編

公司統一編號：**12345678**（所有報支發票必須打此統編）

## 報支科目與標準

### 誤餐費

- **每人每餐上限**：新台幣 150 元（含稅）
- 同一日同一人最多報支 2 餐（午餐、晚餐）
- **酒類不得列誤餐費**
- 與廠商/客戶聚餐 → 改列「交際費」

### 差旅費 — 交通

| 交通工具 | 規則 |
| --- | --- |
| **高鐵** | 限**標準車廂**；商務車廂不予報支（總經理／副總以上特准除外） |
| **台鐵** | 自強號（含）以下，實報實銷 |
| **計程車** | 僅限（1）攜大型物件（2）夜間 22:00 後加班（3）大眾運輸不可達；**單趟上限 500 元** |
| **自行開車** | 里程津貼每公里 4 元，附 Google Maps 路線截圖 |
| **客運／捷運／公車** | 實報實銷 |

### 差旅費 — 住宿

- 國內：每人每晚上限新台幣 2,500 元
- 國外：依分級表（另訂）

### 辦公用品費（核決權限）

| 單筆金額 | 核決層級 |
| --- | --- |
| 500 元以下 | 直屬主管 |
| 501 ~ 3,000 元 | 部門主管 |
| 3,001 元以上 | **事前簽呈** + 總經理核准 |

### 交際費

- **每人每餐上限**：新台幣 800 元
- 須附參與名單（含外部人員姓名與公司）
- 贈禮單筆上限 2,000 元
- **內部員工聚餐不屬於交際費**（年度尾牙、季度團建例外）
- **酒類限交際費**，且須事前申請

### 設備維護費

- 單筆 5,000 元以上：須事前書面報價單 + 部門主管核准

### 教育訓練費

- 報名費／教材費：**事前簽呈**
- 訓練後 14 日內提交「訓練心得報告」

### 雜項支出

- 單筆不得超過 1,000 元

## 報帳期限

- 費用發生後 **30 日內**完成報帳
- **每月 25 日**為當月截止日；逾期列入次月
- 跨年度：1 月 10 日前完成

## 憑證規範

- 統一發票須打公司統編 **12345678**
- 發票破損、模糊、字跡不清 → 不予受理
- 電子發票：須附 QR Code 截圖
- 收據須載明：日期、商家、品項、金額、商家統編／負責人簽章

## 核決權限總表（依單筆金額）

| 報支金額（單筆） | 核決層級 |
| --- | --- |
| 1,000 元以下 | 直屬主管 |
| 1,001 ~ 5,000 元 | 部門主管 |
| 5,001 ~ 30,000 元 | 部門主管 + 財會經理 |
| 30,001 ~ 100,000 元 | 總經理 |
| 100,001 元以上 | 總經理 + 董事長 |
