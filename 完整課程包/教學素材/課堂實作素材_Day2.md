## 素材 1｜海報文案範例對照（示範 1-3 用）

**用途**：講師示範海報製作時，先讓學員看「廢話文案 vs 用 AI 改寫後文案」的差異，再進入 Nano Banana 出圖環節。

### A. 廢話文案範例（Before）

> **暖光咖啡 西西里檸檬咖啡 即將推出**
>
> 親愛的顧客您好：
> 暖光咖啡為了帶給您更好的服務體驗，
> 我們即將推出夏日限定新品「西西里檸檬咖啡」，
> 敬請期待，更多精采內容即將揭曉，
> 歡迎您屆時親臨我們的店面光臨指教。
>
> 暖光咖啡 敬上

**為什麼是廢話文案**（給講師講解用）：
1. 「即將推出」「敬請期待」「精采揭曉」三大廢話齊發
2. 沒講為什麼這杯值得喝（無賣點）
3. 沒給上市日期（無 CTA 時間點）
4. 自稱「敬上」過於官腔

### B. AI 改寫後文案（After，3 組備選）

#### 組 1（清爽夏日感）
> **主標**：盛夏，倒進一片檸檬的清醒
> **副標**：7/1 上市｜西西里檸檬咖啡｜手沖義式 × 義大利檸檬皮油

#### 組 2（文青慢調）
> **主標**：當咖啡遇見地中海的太陽
> **副標**：7/1 暖光咖啡新品｜限量 100 杯／日

#### 組 3（直球行銷）
> **主標**：太熱了，這杯比冷氣有用
> **副標**：7/1 起 西西里檸檬咖啡｜首週買一送一

### C. 海報內容必備資訊清單（給學員檢核用）

```
□ 品牌名（暖光咖啡）
□ 商品名（西西里檸檬咖啡）
□ 上市日期（7/1）
□ 賣點（1 個就好）
□ 優惠資訊（若有）
□ 行動呼籲（CTA）
□ 店家地點 / IG / LINE 任一聯繫管道
```

## 素材 2｜投影片大綱範例（示範 1-4 用）

**用途**：示範用 Gemini Canvas 產出 8 頁投影片之大綱原始素材。

**情境**：綠野選物 春季新品「香氛蠟燭」門市銷售訓練 8 頁簡報。

### 講師應給 Gemini 的「大綱」原始稿

```
主題：春季新品香氛蠟燭門市銷售訓練
受眾：3 家門市同仁，總共 9 人
時間：30 分鐘訓練
品牌核心價值：GREEN（Genuine 真誠 / Respect 尊重 / Excellence 卓越 / Eco 永續 / Network 共創）

預期 8 頁結構（Why-What-How）：

Why（背景）：
P1 封面：「春季新品香氛蠟燭門市銷售訓練」
P2 為什麼香氛蠟燭是本季主打？
   - 春季購物節銷售缺口
   - 客群擴張（25-40 歲女性 + 男性送禮族）
   - 客單價提升空間（單品 NT$680~1,280）

What（解法 / 提案）：
P3 商品介紹（3 款蠟燭）
   - 春日綠蔭（檸檬草 + 雪松）
   - 海岸薰風（海鹽 + 鼠尾草）
   - 暖陽木質（檀香 + 香根草）
P4 各款定位與目標客群
P5 三大銷售切入點
   - 自用：放鬆儀式感
   - 送禮：包裝與卡片服務
   - 居家：搭配陶器擺設
P6 常見客戶 Q&A 預演
   - 「會不會太刺鼻？」「燃燒時間多久？」「孕婦能用嗎？」

How（行動 / 下一步）：
P7 銷售挑戰：每店每週至少 5 組成交
P8 結尾：完訓後一週內回報三個自己會說的銷售切入點

風格要求：
- 主色：淺綠 + 米白（綠野選物品牌色）
- 留白多、字級偏大（50 歲店員也能看清楚）
- 每頁 3 個 bullet 為原則
- 第一頁封面、第八頁結尾頁
```

### 預期投影片內容（給講師對照 AI 產出）

| 頁次 | 標題 | 重點 bullet |
| --- | --- | --- |
| P1 | 春季新品香氛蠟燭門市銷售訓練 | 副標：3/15 上市｜全 3 店同步｜訓練主講：Joyce |
| P2 | 為什麼這季主打蠟燭？ | 客群擴張・客單價提升・購物節缺口 |
| P3 | 三款蠟燭速覽 | 春日綠蔭・海岸薰風・暖陽木質 |
| P4 | 客群與定位 | 自用女性・送禮男性・送長輩 |
| P5 | 三大銷售切入點 | 儀式感・送禮包裝・居家搭配 |
| P6 | 客戶 Q&A 預演 | 燃燒時間・孕婦適用・刺鼻疑慮 |
| P7 | 本週挑戰 | 每店 5 組成交・互助分享話術 |
| P8 | 完訓回報 | 一週內回報 3 個切入點・主管面談 |

## 素材 3｜影片腳本範例（示範 2 用）

**用途**：示範用 Flow 產出 16 秒短片之原始腳本。**雙片段拼接：8 秒 hook + 8 秒展示+CTA**。

**情境**：暖光咖啡 西西里檸檬咖啡 IG Reels 宣傳。

### 完整腳本與分鏡

```yaml
影片名稱: 暖光咖啡｜西西里檸檬咖啡 7/1 上市
時長: 16 秒
規格: 9:16 直式（IG Reels / TikTok）
結構: HSC（Hook 0-3 秒 / Show 3-13 秒 / Call 13-16 秒）

# ============================================
# 鏡頭 1：Hook + Show 開場（8 秒）
# ============================================
鏡頭1:
  時段: 0-8 秒
  關鍵幀風格: |
    一杯透明玻璃杯西西里檸檬咖啡特寫，杯身結露，
    檸檬片懸浮在咖啡與蘇打交界處，逆光，
    背景虛化的木質吧台，極簡攝影風格，
    色調：金色陽光 + 檸檬黃 + 深焙咖啡褐
  動作描述: |
    0-1 秒：黑底，一句白字標題「太熱了。」淡出
    1-3 秒：鏡頭拉近，一片檸檬從上方緩慢落入杯中（Hook 視覺衝擊）
    3-8 秒：鏡頭環繞俯拍，咖啡與蘇打的漸層流動，畫面右下角浮現品名

# ============================================
# 鏡頭 2：Show 收尾 + Call（8 秒）
# ============================================
鏡頭2:
  時段: 8-16 秒
  關鍵幀風格: |
    暖光咖啡店內吧台場景，成品擺盤於原木板上，
    旁邊是檸檬切片與咖啡豆，
    背景柔焦的店長手部動作（不見人臉，避免一致性問題），
    光線：午後窗光 + 暖黃燈光
  動作描述: |
    8-12 秒：鏡頭從成品緩慢推進，焦點轉到玻璃杯
    12-13 秒：店長手輕觸杯緣，鏡頭往上微搖
    13-16 秒：畫面定格於成品，浮現文字 CTA
              「7/1 起｜暖光咖啡 信義店｜限量 100 杯／日」

# ============================================
# 字幕設計
# ============================================
字幕:
  風格: 半透明黑底白字 + 細體繁中
  位置: 下方 1/4 處
  逐句時間軸:
    - 0-1 秒: 「太熱了。」
    - 4-7 秒: 「西西里檸檬咖啡 7/1 上市」
    - 13-16 秒: 「暖光咖啡 信義店 限量 100 杯／日」

# ============================================
# 配樂
# ============================================
配樂:
  風格: Lo-Fi Chill / Bossa Nova
  情緒: 慵懶、夏日、療癒
  音量: 主音 -8dB、靜音 0-1 秒給標題呼吸感
  Flow 內建: 推薦「Summer Stroll」或「Citrus Garden」

# ============================================
# 輸出設定
# ============================================
輸出:
  格式: MP4
  解析度: 1080 x 1920（9:16）
  幀率: 30fps
  最終長度: 16 秒
  發佈管道: Instagram Reels、TikTok
```

## 素材 4｜下午影片實作的「品牌主視覺」備援照片

**用途**：學員若忘記帶自家品牌主視覺照片，講師需提供備援照。

### 暖光咖啡 備援素材（請事先準備檔案）

| 檔名 | 內容 | 用途 |
| --- | --- | --- |
| `warm_facade.jpg` | 暖光咖啡店面外觀（白天） | 場景照 |
| `warm_bar.jpg` | 吧台特寫（無人物） | 製程照 |
| `warm_logo.png` | 品牌 LOGO（透明背景） | 品牌素材 |

### 綠野選物 備援素材

| 檔名 | 內容 | 用途 |
| --- | --- | --- |
| `green_storefront.jpg` | 永康店店面（含招牌） | 場景照 |
| `green_product.jpg` | 香氛蠟燭擺設特寫 | 商品照 |
| `green_logo.png` | 品牌 LOGO（透明背景） | 品牌素材 |

> **講師備忘**：以上檔案為虛構，授課前請依實際品牌另行拍攝授權圖檔；或使用 Unsplash / Pexels 等免費商用素材庫之同類照片替代。
