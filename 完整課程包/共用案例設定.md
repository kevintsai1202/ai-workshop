# 課程共用案例設定

> 本檔案為 4 天課程共用的虛構品牌與角色設定。所有講師示範與學員實作，皆以此為情境基礎。學員可依自身產業二選一帶入。

## 案例品牌

### 案例 A｜暖光咖啡（餐飲｜單店咖啡廳）

| 欄位 | 內容 |
| --- | --- |
| 品牌名稱 | 暖光咖啡 Warm Light Coffee |
| 業態 | 單店獨立咖啡廳，午餐輕食 + 手沖咖啡 |
| 地點 | 台北市信義區 |
| 員工 | 全職 3 人、兼職 5 人 |
| 主要客群 | 鄰近上班族、假日週邊住戶 |
| 行政痛點 | 員工排班公告、月結進貨發票、會議紀錄、Google 評論回覆 |

### 案例 B｜綠野選物（零售｜小型生活選品店）

| 欄位 | 內容 |
| --- | --- |
| 品牌名稱 | 綠野選物 GreenField Select |
| 業態 | 3 家門市的生活選物零售（餐廚、香氛、文具） |
| 地點 | 雙北 3 家門市 |
| 員工 | 總部 4 人、門市每店 2~3 人 |
| 主要客群 | 25-40 歲女性、送禮族 |
| 行政痛點 | 多店進貨單彙整、新品上架公告、教育訓練手冊、客訴處理 |

## 學員作品累積機制

讓學員每天的作品在隔日仍可被引用，逐步堆疊出個人化的「行政 AI 工具箱」：

- **Day 1 產出**：個人提示詞庫（公告、通知、會議紀錄各 1 份）
- **Day 2 產出**：海報草稿、投影片初稿、60 秒宣傳腳本
- **Day 3 產出**：會議紀錄範本、FAQ 初稿、發票 Excel 範本
- **Day 4 產出**：個人化 Agent Skill（將 Day 1~3 的提示詞封裝成可重用技能）
